# my-dotnet-console-app

This is a simple .NET console application.

## Getting Started

To build and run this application, follow these steps:

1. **Prerequisites**: Ensure you have the .NET SDK installed on your machine. You can download it from the official .NET website.

2. **Clone the repository**:
   ```
   git clone <repository-url>
   cd my-dotnet-console-app
   ```

3. **Build the application**:
   ```
   dotnet build
   ```

4. **Run the application**:
   ```
   dotnet run
   ```

## Project Structure

- `Program.cs`: The entry point of the application where the execution begins.
- `my-dotnet-console-app.csproj`: The project file containing configuration settings for the application.
